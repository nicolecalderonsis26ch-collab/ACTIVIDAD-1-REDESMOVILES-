<H1> RESDES MOVILES </H1>
<P> Las redes móviles son infraestructuras de comunicación inalámbrica que permiten la transmisión de voz y datos entre dispositivos (como smartphones y tablets) a través de ondas de radio. Funcionan mediante una red de celdas geográficas, cada una con antenas o estaciones base que se conectan al sistema telefónico, posibilitando la movilidad del usuario sin perder la conexión.Características y Funcionamiento Clave:Infraestructura Celular: El territorio se divide en celdas, cada una cubierta por una torre (BTS o estación base). Al moverte, tu dispositivo cambia de antena sin interrumpir el servicio.Conexión Inalámbrica: Utilizan ondas de radio de baja potencia para vincular tu teléfono con la torre más cercana.Red Central: Las antenas se conectan a una red central que gestiona el tráfico y conecta con internet o redes telefónicas fijas (RTPC).Evolución Tecnológica: Han evolucionado desde la primera generación (1G, solo voz analógica) hasta las redes actuales como 4G y 5G, que ofrecen alta velocidad de datos</P>
